# Changelog

Все значимые изменения в проекте будут документированы в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и этот проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

## [1.0.0] - 2026-01-03

### Добавлено
- Первая версия конвертера тем
- Поддержка всех 362 параметров Desktop темы
- Парсинг Material Design 3 цветовой палитры (n1_*, n2_*, a1_*, a2_*, a3_*)
- Поддержка различных форматов цветов (hex, ARGB, переменные)
- Автоматическое создание фонового изображения
- Создание градиентного фона с акцентными элементами
- Функция adjust_brightness для модификации цветов
- Полная документация и примеры использования

### Поддерживаемые форматы цветов
- Hex: `#1a1a1a`
- ARGB hex: `-14643754h`
- Десятичные: `2147483647`
- Material Design: `n1_900`, `a1_200`
- С прозрачностью: `n1_800(a=80)`, `primaryDark(l=70)`
- Предопределённые: `mWhite`, `mBlack`, `mGreen500`, etc.

### Известные ограничения
- Фоновое изображение создаётся автоматически (не извлекается из Android темы)
- Некоторые специфичные для Android параметры могут не иметь прямых аналогов
